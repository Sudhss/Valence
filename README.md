# Valence

Valence is an ongoing attempt to build a serious developer tool from first principles.

Not a clone.  
Not a demo.  
Not something rushed to look impressive.

Just a clean, desktop-first editor, and eventually a compiler, built to understand how modern tooling actually works under the hood.

This project is **in progress** and intentionally evolving.
